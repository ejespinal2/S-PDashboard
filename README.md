*Python Files:*
All data is taken from the YFinance library.

SP_Stock_Return_Data.py: This file creates an Excel file that measures the monthly returns of individual stocks in the S&P 500 as well as the yearly returns and yearly volatility of each stock. 

SP_Return_Data.py: This file returns an Excel file that measures monthly returns for the S&P 500 stock as well as Yearly returns and yearly volatility.

*Excel:*
As previously noted, the files sp500_Stock_Returns and sp500_Returns are data derived from the Python files.

Yearly_Performance.xlsx: This file tracks the Yearly returns and Yearly volatility from individual stocks in the S&P500 into one file from the beginning of 2015 to July 2025.

Monthly_Returns.xlsx: This file tracks the monthly returns for individual stocks from 2015 to July 2025.
